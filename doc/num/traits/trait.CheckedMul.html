Performs multiplication that returns None instead of wrapping around on underflow or overflow.

Required Methods

fn checked_mul(&self, v: &Self) -> Option<Self>

Multiplies two numbers, checking for underflow or overflow. If underflow or overflow happens, None is returned.

Example

extern crate num; fn main() { use num::CheckedMul; assert_eq!(5u8.checked_mul(&51), Some(255)); assert_eq!(5u8.checked_mul(&52), None); }
